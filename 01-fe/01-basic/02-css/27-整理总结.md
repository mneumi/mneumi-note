## 脱离文档流操作总结

| 脱离文档流操作     | 说明     |
| ------------------ | -------- |
| float: left        | 左浮动   |
| float: right       | 右浮动   |
| position: absolute | 绝对定位 |
| position: fixed    | 固定定位 |



## 使元素显示模式转为 inline-block的操作

| display转为 inline-block的操作 | 说明     |
| ------------------------------ | -------- |
| float: left                    | 左浮动   |
| float: right                   | 右浮动   |
| position: absolute             | 绝对定位 |
| position: fixed                | 固定定位 |
