## 概述

`nrm`用于方便地设置 `npm` 的镜像源



## 安装

```shell
npm install nrm -g
# OR
yarn global add nrm
```



## 常用命令

| 命令                  | 说明                 |
| --------------------- | -------------------- |
| nrm ls                | 列出所有能用的镜像源 |
| nrm current           | 查看当前使用的镜像源 |
| nrm use [mirror-name] | 更改npm镜像源        |
