## npm包目录结构

一个 `node.js` 模块是基于 `CommonJS` 模块化规范实现的，严格按照 `CommonJS` 规范，模块目录下除了必须包含包描述文件 `package.json` 以外，还需要包含以下目录

| 目录 | 说明                       |
| ---- | -------------------------- |
| bin  | 存放可执行二进制文件的目录 |
| lib  | 存放js代码的目录           |
| doc  | 存放文档的目录             |
| test | 存放单元测试用例代码的目录 |
