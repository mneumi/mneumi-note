## 路由模式

### 概述

路由模式一共有两种：Hash模式和History模式

### 对比

| 路由模式    | 优点                                             | 缺点                                               |
| ----------- | ------------------------------------------------ | -------------------------------------------------- |
| Hash模式    | 兼容性好，低版本浏览器也能使用<br />无需后端配置 | 路径中有个#，不好看                                |
| History模式 | 路径中无#，好看                                  | 兼容性较差，低版本浏览器不能使用<br />需要后端配合 |
